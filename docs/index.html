There are many ways to get started with the pattern library. It all depends on your project needs, technical understanding and whether you are a developer or a designer. Don’t forget to check out the DTA's community to see what others are doing!

The three main ways of getting started with the pattern library include:

  • npm install the components
  • Download page to install the components
  • Design system starter kit

Once you have the components in a website we have advanced guides for developers on how to:

  • Customise the colour scheme
  • Customise existing components
  • How to use the font size and space functions
  • Set up a development environment

Throughout the process you may notice issues, missing features or have ideas for new components. If you would like to contribute or help build the future of the pattern library you can:

  • Join the community
  • Submit a bug report
  • Submit a feature request
  • Suggest a component
